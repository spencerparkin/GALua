GALua -- A Lua module providing Geometric Algebra (GA) Support

The goal of the GALua module is to endow the Lua programming language with support for GA data types (i.e. multi-vectors.) At the time of this writing, the module is experimental and admittedly slow, but it works. You can obtain a copy of the module as a ZIP archive or tar-ball, along with the entire project source.
