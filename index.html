The GALua Module

Welcome! You've just happened upon the GALua project page! Enjoy your stay!

What is GALua?

GALua is a Lua module written in C++ adding support for GA (geometric algebra). The goal of the GALua module is to endow the Lua programming language with support for GA data-types (i.e. multi-vectors.)

How do I get GALua?

The current, official distribution of GALua is GALua-win32-v1.0.zip. It is compatible with Lua 5.2. As of this writing, there is is not yet a zipped archive for Linux or other platforms, but Linux is on the list of things to do.

Alternatively, you can grab the entire project tree and build GALua yourself. To do that, simply clone the GALua reposirty by issuing the following command. 

 $ git clone https://github.com/spencerparkin/GALua.git

You'll need to know that GALua depends upon a statically linked library known as CalcLib which provides the GA calculation engine. It is kept in a seperate repository, because other projects, (such as GAVisTool), depend upon it too.
